🖨️ Генератор QR-кодов для наклеек

Создайте уникальные QR-коды для ваших дронов

💡 Например: если последний номер был 010, начните с 011

📖 Инструкция

📌 Префиксы артикулов:
• Mavic 3 серия → M3, M3C, M3P, M3PC, M3E, M3T, M3M
• Mavic 2 серия → M2P, M2Z
• Mavic Pro → MP
• Air серия → A3S, A3, A2S, A2
• Mini серия → MN5P, MN4P, MN4K, MN3P, MN3, MN2S, MN2, MNSE
• Neo → NEO2, NEO
• Flip → FLP
• Avata → AVA2, AVA
• Inspire → INS3, INS2
• Matrice → MTR350, MTR30, MTR30T, MTR300

📌 Как продолжить нумерацию:
• Посмотрите последний использованный номер для вашей модели
• В поле "Начать с номера" укажите следующий номер
• Сгенерируйте и распечатайте новые наклейки

🖨️ Печать:
• Нажмите "Печать наклеек"
• В диалоге печати выберите "Фактический размер"
• Масштаб: 100%
