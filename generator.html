🖨️ Генератор QR-кодов для наклеек

Создайте уникальные QR-коды для ваших товаров.

📖 Инструкция

1. Создание наклеек:
• Выберите модель и количество
• Нажмите "Сгенерировать QR-коды"
• Нажмите "Печать наклеек"

2. Формат "Термопринтер (58x40мм):
• Размер каждой наклейки 58x40 мм
• Подходит для принтеров: Zebra, Argox, TSC

3. Формат "A4":
• 9 наклеек на листе (3x3)
• Для обычного принтера

4. Формат QR-кода:
• MINI2-001, MINI2-002 ... для Mini 2
• AIR2S-001, AIR2S-002 ... для Air 2s
