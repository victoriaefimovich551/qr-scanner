📌 Как продолжить нумерацию:
• Посмотрите последний использованный номер
• В поле "Начать с номера" укажите следующий номер
• Сгенерируйте и распечатайте новые наклейки

📌 Пример:
• У вас уже есть наклейки MINI2-001 ... MINI2-050
• Нужно ещё 20 наклеек
• Укажите: Начать с номера = 51, Количество = 20
• Получите: MINI2-051 ... MINI2-070

📖 Инструкция

Мгновенная печать:
• QR-коды генерируются через API qrserver.com
• При печати используется тот же API — без лишних преобразований
• Страница печати открывается в новой вкладке (разрешите всплывающие окна)

Формат "Термопринтер (58x40мм):
• Размер каждой наклейки 58x40 мм
• В диалоге печати выберите "Фактический размер"
• Масштаб: 100%

Формат "A4":
• 9 наклеек на листе (3x3)
• Для обычного принтера
