🖨️ Генератор QR-кодов для наклеек

Создайте уникальные QR-коды для ваших товаров. Каждый код будет содержать уникальный идентификатор модели.

📦 Пакетная печать

Создайте QR-коды для нескольких моделей сразу

📖 Инструкция по использованию

1. Создание наклеек:
• Выберите модель и количество
• Нажмите "Сгенерировать QR-коды"
• Распечатайте на самоклеящейся бумаге
• Вырежьте и наклейте на товары

2. Формат QR-кода:
Каждый QR-код содержит уникальный текст вида: MINI2-001, MINI2-002 и т.д.
При сканировании приложение автоматически:
• Распознаёт модель (MINI2 → Mini 2)
• Создаёт артикул (MN2-001)
• Добавляет товар в таблицу

3. Приёмка товаров:
• Откройте основное приложение
• Перейдите на вкладку "➕ Приёмка"
• Сканируйте QR-код с наклейки
• Укажите цену (если нужно)
• Товар автоматически добавится в таблицу

4. Поиск товара:
• Сканируйте тот же QR-код
• Приложение покажет всю информацию о товаре

💡 Совет: Используйте самоклеящуюся бумагу для наклеек или обычную бумагу + прозрачный скотч.
